# HTML Demo Workspace

This repository is a lightweight HTML and CSS learning workspace. It keeps the notes and practice exercises available while giving the main demo site a cleaner home under `site/`.

It is related to the final project from the same WBS HTML/CSS Weiterbildung:
[eichhoernchen-wiki-projektarbeit](https://github.com/lylehmann/eichhoernchen-wiki-projektarbeit).
This workspace is the practice and preparation side of that course, while the GitHub repository is the final course deliverable.

## Project Structure

- `site/` contains the main runnable demo pages.
- `exercises/` contains smaller lesson and task files for practice.
- `docs/` contains Markdown notes and reference material.
- `docs/course-context.md` explains how this workspace connects to the final project.

## Learning Progression

This repo now reflects a clearer course progression:

1. Read notes in `docs/` to review HTML and CSS concepts.
2. Practice those concepts in `exercises/`.
3. Use `site/` as the clearest small end-to-end example.
4. Apply the same ideas in the final squirrel website project on GitHub.

## How to Use It

Open [site/index.html](site/index.html) in a browser to view the main demo.

You can also open individual exercise files directly:

- [exercises/lesson-d1/index.html](exercises/lesson-d1/index.html)
- [exercises/task-d1/contact-form.html](exercises/task-d1/contact-form.html)
- [exercises/task-d1/sport-selection-form.html](exercises/task-d1/sport-selection-form.html)
- [exercises/task-d1/table.html](exercises/task-d1/table.html)

For quick local testing, you can either:

- open the HTML files directly in your browser, or
- use a simple local server such as the VS Code Live Server extension.
