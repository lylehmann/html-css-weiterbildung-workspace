# Course Context

This workspace belongs to the same HTML/CSS Weiterbildung as the final GitHub project
[`eichhoernchen-wiki-projektarbeit`](https://github.com/lylehmann/eichhoernchen-wiki-projektarbeit).

## Relationship to the Final Project

- This repository is the practice workspace.
- The GitHub repository is the final project submission.
- Both belong to the same course and build on the same HTML/CSS fundamentals.

## How the Structure Maps to the Course

- `docs/` contains notes about core topics such as CSS, the box model, flexbox, media queries, and project tasks.
- `exercises/` contains smaller practice files for forms, tables, selectors, layout basics, and simple styling.
- `site/` contains a more coherent mini demo that sits between the raw exercises and the final project.

## Concepts That Carry Over to the Final Project

- semantic HTML page structure using elements like `header`, `nav`, `main`, `section`, `article`, and `footer`
- external CSS files for shared styling across multiple pages
- consistent navigation between pages
- forms, tables, text content, images, and general content layout
- basic responsive thinking and cleaner visual organization

## Suggested Interpretation

This repo is best understood as the learning path that supports the final project:

1. notes and exercises to learn the building blocks
2. a small demo site to practice combining them
3. the final squirrel website as the polished course outcome
